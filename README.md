# Dubbo-gomall
### 🚀 项目概览  
Dubbo-gomall 是一个利用Dubbo-go微服务框架和seata-go分布式事务框架以及dubbo-go-pixiu高性能网关实现的一个分布式电商系统，除此之外，Dubbo-gomall还基于Eino框架实现了智能客服服务，包括RAG，MCP，muitl-Agent等等技术。
